## CSS @rules (at-rules)

### `@import "styles2.css";`

### `@media`
- used to create media queries
    - use conditional logic for applying CSS styling
- `css@media (min-width: 30em) {body {background-color: blue;}}`
    - defines a blue background if the browser viewport is wider than 30em
